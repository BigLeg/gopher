# Mapreduce for inverted index

## Prerequisites
1. Set up a cluster or a pseudo-distributed environment.
2. Add `$HADOOP_HOME/bin` to `PATH` in `~/.bash_profile`

## Instructions

To build:

```
./gradlew
```

To run:

```
./gradle run
```
